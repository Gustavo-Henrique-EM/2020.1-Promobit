    Protótipo

    Introdução

    Um protótipo é uma maneira de manifestar um design de forma em que o stakeholder possa interagir e explorar o mesmo, além disso é possível validar os requisitos que foram elicitados através dele, através disso, se torna uma forma interessante de validar os requisitos do projeto.

    Protótipo

    Nós fizemos um protótipo para alidar aalguns requisitos que elicitamos e modelamos durante o projeto. O protótipo tem um objetivo bem simples, porém direto, fizemos apenas algumas telas para mostrar a validade dos requisitos. O protótipo foi realizado no Figma, e pode ser acessado aqui.
