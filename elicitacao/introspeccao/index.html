    Introspecção

    Metodologia

    A introspecção é uma técnica reflexiva de elicitação de requisitos. Baseia-se em imaginar que tipo de sistema um usuário, ou no nosso caso uma persona, iria querer se estivesse executando esta tarefa, utilizando este equipamento, entre outros. Ou seja, imaginar que propriedades um sistema deveria possuir de forma a obter sucesso.
