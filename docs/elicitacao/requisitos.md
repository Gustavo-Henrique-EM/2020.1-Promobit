# Todos os Requisitos

## Notação

Nós decidimos utilizar a seguinte notação para a identificação dos requisitos:

- **RF** *NÚMERO*: para os requisitos funcionais. Exemplo: RF3
- **RF** *NÚMERO*: para os requisitos não funcionais. Exemplo: RFN42
